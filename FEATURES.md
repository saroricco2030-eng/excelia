# Excelia - 전문 엑셀 편집 및 뷰어

![Excelia](https://img.shields.io/badge/version-1.0.0-blue) ![Flutter](https://img.shields.io/badge/Flutter-3.10.8+-green) ![License](https://img.shields.io/badge/license-MIT-blue)

## 📋 개요

**Excelia**는 Polaris 디자인 기반의 전문적인 엑셀 편집 및 뷰어 애플리케이션입니다. 
Windows, macOS, Linux, iOS, Android 등 다양한 플랫폼을 지원합니다.

## 🎨 주요 특징

### 핵심 기능
- ✅ **엑셀 파일 열기** - .xlsx, .xls 형식 지원
- ✅ **엑셀 파일 편집** - 셀 값 수정 및 추가
- ✅ **엑셀 파일 저장** - 변경사항 저장
- ✅ **다중 시트 지원** - 여러 시트 관리
- ✅ **새 파일 생성** - 빈 엑셀 파일로 시작

### UI 특징 (Polaris 디자인)
- 🎯 **전문적인 인터페이스** - Microsoft Office 스타일
- 🔵 **Microsoft Blue 컬러** - 브랜드 일관성
- 📊 **스프레드시트 그리드** - 행/열 헤더 포함
- 🛠️ **포맷팅 도구바** - 굵게, 기울임, 밑줄, 색상 (예정)
- 📑 **시트 탭** - 시트 전환 및 추가

### 데이터 관리
- 📁 **파일 관리** - 파일 열고/저장하기
- 📝 **셀 편집** - 다이얼로그를 통한 직관적 편집
- 📊 **데이터 표시** - 행/열 구분자와 함께 표현

## 🚀 사용 방법

### 1. 애플리케이션 실행
```bash
cd c:\Users\saror\Desktop\excelia
flutter run
```

### 2. 새 파일 만들기
- 시작 화면에서 "새 파일 만들기" 버튼 클릭
- 또는 상단 메뉴에서 "파일 열기" 아이콘 클릭

### 3. 파일 열기
- 상단 메뉴에서 📁 아이콘 클릭
- .xlsx 또는 .xls 파일 선택
- 자동으로 스프레드시트가 로드됨

### 4. 셀 편집
- 셀을 클릭하면 편집 다이얼로그 열림
- 값을 입력하고 "저장" 클릭
- 자동으로 UI에 반영됨

### 5. 파일 저장
- 상단 메뉴에서 💾 아이콘 클릭
- 사용자 Documents 폴더에 자동 저장
- `excelia_[timestamp].xlsx` 형식으로 저장

### 6. 시트 관리
- 하단 시트 탭에서 시트 선택
- "+" 아이콘으로 새 시트 추가
- 시트 이름 "시트1", "시트2" 등으로 자동 생성

## 🎯 아키텍처

### 상태 관리 (Provider)
- `ExcelProvider`: 엑셀 데이터 및 상태 관리
  - `excel`: 현재 로드된 Excel 객체
  - `currentSheet`: 현재 선택된 시트
  - `filePath`: 현재 파일 경로

### 주요 위젯
- `MyApp`: 루트 위젯 및 테마 설정
- `MyHomePage`: 메인 화면 및 레이아웃
- `SpreadsheetViewer`: 스프레드시트 표시 및 편집
- `ExcelProvider`: 비즈니스 로직
