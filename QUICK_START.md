# 🚀 Excelia 빠른 시작 가이드

## 📌 5분 안에 시작하기

### 1단계: 애플리케이션 실행 (1분)
```bash
cd c:\Users\saror\Desktop\excelia
flutter run
```

### 2단계: 파일 만들거나 열기 (1분)
- **새 파일 만들기**: 화면 중앙의 "새 파일 만들기" 버튼 클릭
- **기존 파일 열기**: 상단의 📁 "파일 열기" 아이콘 클릭

### 3단계: 데이터 입력 (2분)
- 셀을 클릭하여 편집 다이얼로그 열기
- 값 입력 후 "저장" 클릭
- 원하는 만큼 반복

### 4단계: 파일 저장 (1분)
- 상단의 💾 "파일 저장" 아이콘 클릭
- 자동으로 Documents 폴더에 저장

## 💡 사용 팁

### 팁 1: 빠른 데이터 입력
셀을 클릭 → 다이얼로그에 입력 → 저장 → 다음 셀 → 반복

### 팁 2: 여러 시트 관리
- 하단 탭에서 시트 클릭하여 전환
- "+" 아이콘으로 새 시트 추가
- 시트명은 자동으로 "시트1", "시트2" 등으로 지정

### 팁 3: 대량 데이터 작업
1. 엑셀에서 미리 데이터 준비
2. 파일로 저장
3. Excelia에서 엽니다
4. 필요시 수정 후 저장

### 팁 4: 파일 위치
저장된 파일:
```
C:\Users\[사용자명]\Documents\excelia_[시간].xlsx
```

## 🎯 일반적인 작업

### 작업 1: 판매 데이터 입력
```

B열: 가격
C열: 수량
D열: 합계 (= B * C)

### 작업 2: 고객 목록 생성
```

B열: 고객명
C열: 전화번호
D열: 이메일

```

```

## ❓ FAQ

**Q: 여러 개의 파일을 동시에 열 수 있나요?**
A: 현재 버전에서는 한 번에 하나의 파일만 열 수 있습니다. 다른 파일을 열면 이전 파일은 닫힙니다.

**Q: 저장된 파일의 형식은?**
A: 모든 파일은 표준 .xlsx (엑셀 2007+) 형식으로 저장됩니다.

**Q: 실행 취소 기능은?**
A: 현재 버전에는 미포함되어 있습니다. (예정 기능)

**Q: 매우 큰 파일을 열 수 있나요?**
A: 메모리에 따라 제한됩니다. 10,000행 이상은 성능 저하가 있을 수 있습니다.

**Q: 수식은 지원하나요?**
A: 현재 버전에서는 일반 텍스트만 저장됩니다. 수식 지원은 향후 계획입니다.

## 🔧 문제 해결

### 문제: 파일을 열 수 없음
**해결책:**
1. 파일이 .xlsx 또는 .xls 형식인지 확인
2. 파일이 손상되지 않았는지 Excel에서 확인
3. 최신 버전의 Excelia 사용

### 문제: 저장이 실패함
**해결책:**
1. Documents 폴더에 쓰기 권한이 있는지 확인
2. 디스크 공간 부족 확인
3. 아무 파일도 로드된 파일이 없는지 확인

### 문제: 앱이 느림
**해결책:**
1. 열려있는 큰 파일 크기 확인
2. 필요 없는 시트 삭제
3. 앱 재시작 (핫 리스타트)
